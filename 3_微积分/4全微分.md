# 偏导数(方向导数)

    函数在某一个方向上的导数

    在数学中，一个多变量的函数的偏导数，
    就是它关于其中一个变量的导数而保持其他变量恒定（相对于全导数，在其中所有变量都允许变化）

# 梯度

    梯度是全部偏导数 向量的结果
  
    梯度的本意是一个向量（矢量）
    表示某一函数在该点处的方向导数沿着该方向取得最大值，
    即函数在该点处沿着该方向（此梯度的方向）变化最快，变化率最大（为该梯度的模）。
